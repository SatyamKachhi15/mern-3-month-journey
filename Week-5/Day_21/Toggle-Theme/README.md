This project implements a global Light/Dark theme using React Context API. I created a ThemeContext using createContext() and wrapped the application with ThemeProvider. Inside the Provider, I managed the theme using useState and created a custom toggleTheme() function to switch between light and dark modes. The Provider shares both the current theme and the toggle function with all child components. Components like Navbar, Home, and ThemeToggle access these values using useContext(), eliminating the need for prop drilling. Whenever the user toggles the theme, the state updates, the Provider supplies the new value, and all consuming components automatically re-render with the updated theme.
