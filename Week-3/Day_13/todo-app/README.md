## React To-Do App

### Project Overview

Built a To-Do application using React that allows users to add and delete tasks dynamically.

### Features

* Add new tasks
* Delete existing tasks
* Dynamic list rendering

### Concepts Practiced

* React Components
* useState Hook
* Controlled Components
* Controlled Inputs
* Forms in React
* Form Submission
* onChange Event
* onSubmit Event
* preventDefault()
* Rendering Lists using map()
* key Prop
* Array Methods (map, filter)
* Event Handling
