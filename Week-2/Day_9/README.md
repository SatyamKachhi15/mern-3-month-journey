## To-Do App with Local Storage

### Features

* Add Tasks
* Delete Tasks
* Mark Tasks as Complete
* Undo Completed Tasks
* Due Date Support
* Pending Tasks Section
* Completed Tasks Section
* Persistent Storage using localStorage
* Auto Load Tasks on Refresh

### Concepts Practiced

* DOM Manipulation
* Event Handling
* Arrays and Objects
* localStorage
* JSON.stringify()
* JSON.parse()
* Dynamic Rendering
