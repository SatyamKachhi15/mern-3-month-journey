# Week 2 - Day 8

## Product List App

### Objective

Build a Product List Application using JavaScript Arrays and Objects.

### Features Implemented

* Product data stored as an array of objects
* Search products by name
* Filter products by category
* Sort products by price
* Grid View and List View toggle
* Dynamic rendering using JavaScript DOM manipulation

### Deployment

* Source code pushed to GitHub
* Application deployed on Vercel
* Live Demo: https://product-listing-app-woad.vercel.app/

### Learning Outcomes

* Arrays and Objects in JavaScript
* Array Methods (filter, sort, forEach, map)
* Object Property Access
* DOM Manipulation
* Event Handling
* Dynamic UI Rendering
* Vercel Deployment Workflow
* Git and GitHub Workflow

### Additional Learning

* Revised React Hooks concepts:

  * useState
  * useEffect
  * useRef
  * useContext
  * Custom Hooks (Introduction)
