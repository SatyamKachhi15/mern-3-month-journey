# Product List App

## Objective

Build a Product List Application using Arrays, Objects, DOM Manipulation, and Event Handling.

## Features

* Display Products
* Search by Product Name
* Filter by Category
* Sort by Price
* Grid View
* List View

## Concepts Practiced

* Arrays
* Objects
* Object Property Access
* forEach()
* filter()
* sort()
* DOM Manipulation
* Event Handling
* Render Functions
